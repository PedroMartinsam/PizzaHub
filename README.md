## 📖 Sobre o Projeto

O **PizzaHub** é um template premium desenvolvido para pizzarias que desejam fortalecer sua presença digital e aumentar as vendas sem depender de marketplaces.

O cliente navega pelo cardápio, escolhe tamanhos, bordas, adicionais, monta o carrinho e finaliza o pedido diretamente pelo WhatsApp da pizzaria.

O objetivo é proporcionar uma experiência simples para o cliente e fácil de administrar para o proprietário.

## 🚀 Como executar o projeto

### Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/PizzaHub.git
```

### Entre na pasta

```bash
cd PizzaHub
```

### Instale as dependências

```bash
npm install
```

### Execute

```bash
npm run dev
```

Abra:

```text
http://localhost:3000
```

## 🎯 Objetivo

O PizzaHub foi criado para permitir que uma pizzaria tenha um site profissional com pedidos via WhatsApp de forma simples, rápida e elegante.

O projeto foi pensado para ser reutilizado em diferentes clientes, bastando alterar informações como:

* Logo
* Nome
* Cardápio
* WhatsApp
* Cores
* Endereço
* Redes sociais
